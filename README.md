# vue-automation-admin

## 这是什么

一款开箱即用的 Vue 中后台管理系统模版，基于 [vue-automation](http://eoner.gitee.io/vue-automation) & [Element](https://element.eleme.cn)

## 特点

- 支持 [vue-automation](http://eoner.gitee.io/vue-automation) 全部特性
- 一键切换布局，轻松更换主题
- 依据路由轻松配置导航栏
- 支持全方位权限验证
